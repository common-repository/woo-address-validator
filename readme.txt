== Description ==

Global Address Validation for WooCommerce

The WooCommerce Address Validator plugin uses the Byteplant Address-Validator API (https://www.byteplant.com/address-validator/) to validate and correct billing/shipping/delivery addresses for 240+ countries world-wide. Stop wasting time and money with undelivered packages or having to reach out to customers to get the correct addresses.

= Supported Plugins / Compatible Forms =

The plugin integrates seamlessly with:

* WooCommerce

= Key Features =

* Automatic validation and correction of billing and shipping addresses
* Customers get a real-time suggestion if they enter an undeliverable address
* Easy-to-use, automatically integrates with WooCommerce checkout pages

== Installation ==

1. Unzip and upload the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin in the Settings section of the Administrator Dashboard.
4. Register for an API key and get 625 free credits (no credit card required):
https://www.byteplant.com/address-validator/free-trial-registration.html
5. Copy and paste your API key to the Settings section of the Administrator Dashboard.
